    Bellman Ford Revisited

    f(b,k) = length of the shortest path from source node S to B in using <= k edges

    recurrence relationship: f(b,k) = * 0, if b == s and k == 0 * infinity, if b != s and k == 0 * min(f(b, k-1), min(f(v, k-1) + w(v, b))), for all v that is connected to b

    Memory: O(n) if we use bottom up, becuase k only depends on the states of k-1s, so one array will do

    Runtime:

    def ssspBotoomUp(G, s):

    Bellman Ford Lemmas: 1. At all points in time, for all b, T[b] = the length of some s->b path. 2. for all k, after k iterations of the outer loop, for all b, T[b] <= f(b,k)

    All Pairs Shortest Paths

    output should be a 2-D arryaT[1:n][1:n] such that T[i][j] = length of shortest path from i to j

    #Runtime: O(v^3)
def FloydWarshall(G):
    initialized T[1:n][1:n] to infinity
    for i = 1 to n, T[i][i] <- 0
    for all i!=j, T[i][j] = weight(e(i,j))
    for k = 1 to n
        for i = 1 to n:
            for j = 1 to n:
                T[i][j] = min(T[i][j], T[i][k] + T[k][j])

    f(i,j,k) is the length of the shortest path from i to j using intermediate vertices that must be in {1, ..., k}, we want f(*, * , n)

    recurrence: f(i,j,k) = * 0 if i = j, k = 0 * w(i, j) if i != j, k = 0 * min (f(i, j, k-1), f(i, k, k-1) + f(k, j, k-1))

    Longest Increasing Subsequence

    input: Array A output:

    f(last,i) length of LIS in A in which all elements must > A[last] we want: max(f(i, i+1) + 1) for 1<i<n

    recurrence: f(last, i) = * 0 if i > n * f(last, i + 1) if A[i] <= A[last] * max(f(last, i+1), 1+f(i, i + 1))

    Traveling Salesman

    Given a complete weighted Graph G, we want the shortest path starting at 1 to visit all other vertices once

    f(i,s) = the shortest total distance to visit all vertices in S starting at i, we want f(1, {2,...,n})

    recurrence: f(i,s) = * 0 if s = 0 * min(w(i,j) + f(j,S-j)) for all j in S

    Runtime: O(n^2 * 2*n)
